# Images Folder

Place your images here according to the structure outlined in `IMAGE_STRUCTURE.md`.

## Quick Reference

### Root Level
- `headshot.jpg` - Hero section profile picture
- Label logos: `island-logo.jpg`, `pre-logo.jpg`, `motorsport-logo.jpg`, `empire-logo.jpg`

### Timeline Logos
Place in `timeline/` folder:
- `northeastern-logo.jpg` - Northeastern University
- `colby-logo.jpg` - Colby College
- `massa-logo.jpg` - Massa Products Corporation
- `udha-logo.jpg` - Uganda Development and Health Associates
- `avinet-logo.jpg` - Avinet Research Supplies
- `soft-tissue-lab-logo.jpg` - Soft Tissue Mechanics Lab
- `qc-project-logo.jpg` - Quality Control Project

### Project Images
Place in `projects/` folder - see `IMAGE_STRUCTURE.md` for complete list.

## File Extensions
The components will automatically try different file extensions (.jpg → .jpeg → .png) if the first one doesn't exist.

For detailed information, see `IMAGE_STRUCTURE.md`.
